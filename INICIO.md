# REIVEL - Plataforma Educativa ECOEMS

## 🚀 INICIO RÁPIDO

### ¿Por dónde empiezo?

**DÍ 1️⃣ a tres formas de abrir la aplicación:**

#### Forma 1: Abrir directamente (Más fácil)
1. Busca la carpeta: `REIVEL.WEB` 
2. Abre el archivo: **`start.html`** (haciendo doble clic)
3. ¡Listo! Aparecerá tu navegador con la aplicación

#### Forma 2: Desde el navegador
1. Abre tu navegador (Chrome, Firefox, etc.)
2. Presiona: **Ctrl+O** (Windows) o **Cmd+O** (Mac)
3. Busca la carpeta `REIVEL.WEB` y selecciona: **`index.html`**
4. ¡Hecho!

#### Forma 3: Arrastrar el archivo
1. Abre tu navegador
2. Arrastra el archivo **`index.html`** directo al navegador
3. ¡La aplicación se abrirá!

## ⚙️ Si algo no funciona

**El archivo no abre:**
- Verifica que los archivos estén en la misma carpeta
- Intenta con otro navegador (Chrome está recomendado)

**La interfaz se ve rara:**
- Presiona Ctrl+0 para resetear el zoom
- Recarga la página (F5)
- Agranda la ventana del navegador

**No aparecen las preguntas:**
- Recarga la página (Ctrl+R)
- Cierra y abre de nuevo el navegador
