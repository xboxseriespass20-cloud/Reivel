# 📖 INSTRUCCIONES PARA USAR REIVEL

## ¿Cómo abrir la plataforma?

### Opción 1: Desde la carpeta del proyecto
1. Navega a la carpeta: `c:\Users\Ritr0x\OneDrive\Documentos\REIVEL.WEB`
2. Abre uno de estos archivos en tu navegador:
   - **start.html** (Recomendado - Panel de inicio)
   - **index.html** (Página principal)
   - **ecoems.html** (Acceso directo a ECOEMS)

3. Para abrir, puedes:
   - Hacer doble clic en el archivo HTML
   - O arrastrar el archivo a tu navegador abierto

### Opción 2: Usando un servidor local
Si tienes Python instalado:
```bash
# Navega a la carpeta del proyecto
cd c:\Users\Ritr0x\OneDrive\Documentos\REIVEL.WEB

# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

Luego abre: http://localhost:8000

### Opción 3: Usando un servidor Node.js
```bash
# Instala http-server globalmente
npm install -g http-server

# Navega a la carpeta y ejecuta
cd c:\Users\Ritr0x\OneDrive\Documentos\REIVEL.WEB
http-server
```

## 🔧 Solución de Problemas

### Las preguntas no aparecen
- Recarga la página (F5 o Ctrl+R)
- Limpia el caché del navegador
- Intenta en otro navegador

### Progreso no se guarda
- Verifica que localStorage esté habilitado
- Revisa que tengas espacio en disco
- Prueba en modo incógnito del navegador

### Interfaz se ve rara
- Ajusta el zoom del navegador (Ctrl+0)
- Amplía la ventana del navegador
- Prueba con otro navegador
